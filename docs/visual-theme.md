# AcadCheck AI Visual Theme

Tema visual saat ini mengikuti arah screenshot: editorial document workspace, warm paper, calm, dan premium.

## Font

### Display / Judul Besar

- Font: `Cormorant Garamond`
- Fallback: `Georgia`, `Times New Roman`, `serif`
- Dipakai untuk: page title, heading utama, judul card/section, brand text, dan angka metric besar.
- Karakter visual: serif editorial, kontras tebal-tipis, terasa seperti arsip/dokumen akademik.

### Body / UI

- Font: `Inter`
- Fallback: `ui-sans-serif`, `system-ui`, `Segoe UI`, `sans-serif`
- Dipakai untuk: paragraf, tabel, form, input, tombol, badge, navigasi, dan isi data.
- Karakter visual: bersih, mudah dibaca, cocok untuk tabel dan workspace.

## Document Library Title

- Font: `Cormorant Garamond`
- Desktop size: `clamp(4.2rem, 7.2vw, 5.7rem)`
- Line height: `0.92`
- Desktop behavior: `white-space: nowrap`
- Color: `#0B261D`

Title dibuat melebar horizontal seperti screenshot, tetapi tidak setinggi/besar versi sebelumnya.
